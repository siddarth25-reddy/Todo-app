# 📝 Todo App

A simple and responsive Todo Application built to help users organize and manage daily tasks efficiently. This project allows users to add, complete, and delete tasks through a clean and user-friendly interface.

## 🚀 Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Responsive user interface
- Real-time task management
- Clean and intuitive design

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)

## 📂 Project Structure

```bash
Todo-app/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## ⚙️ Getting Started

### Clone the Repository

```bash
git clone https://github.com/siddarth25-reddy/Todo-app.git
```

### Navigate to Project Folder

```bash
cd Todo-app
```

### Run the Project

Simply open `index.html` in your browser.

## 🎯 Usage

1. Enter a task.
2. Click the Add button.
3. Mark tasks as completed when finished.
4. Delete tasks when no longer needed.
